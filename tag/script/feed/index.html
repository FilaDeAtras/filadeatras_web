Pues creas un script como el siguiente:

#!/bin/bash
ftp -inv servidor.ftp<<EOF
user usuario password
binary
cd backups
lcd /backups/
put backup_de_hoy.tar.gz
bye
EOF

NOTA: Rsync lo hace mucho mejor en estos casos, pero no se me ocurría otro caso de uso …
