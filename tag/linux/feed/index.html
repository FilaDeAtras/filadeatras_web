A modo de bloc de notas público voy a dejar un par de comandos de linux, muy frecuentes y que habitualmente no recuerdo, como mínimo me servirá para no tener que googlear y mirarlo aquí directamente…

    1. Instalar un paquete deb:
    dpkg -i nombre_del_paquete.deb
    Si da error por dependencias, las resolvemos con:
    apt-get install -f
    2. Instalar las headers del kernel:
    apt-get install linux-headers-$(uname -r)
