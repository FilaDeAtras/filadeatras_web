Hoy os presento un pequeño resumen de ellos:

Empezando por el escritorio, no sé vosotros pero a mi no me gusta eso de tener los botones a la izquierda.

Devolver los botones de ventana a la derecha

  1. Apretamos Alt + F2 y ejecutamos:  gconf-editor
  2. Vamos al apartado apps/metacity/general
  3. Buscamos button_layout y cambiamos el orden a menu:minimize,maximize,close
  4. Aceptamos y listo

Una barra Dock muy cool, Avant Window Navigator + Theme Clearlooks

AWN es un conocido Dock para Gnome, con el que podremos substituir esa maldita barra inferior. Lo bueno que tiene para mí son los temas que podemos ponerle, en especial me encanta el Clearlook. Para instalarlo ponemos en la terminal:

sudo add-apt-repository ppa:albyrock87/lucidoppa

sudo add-apt-repository ppa:awn-testing/ppa && sudo apt-get update

sudo apt-get install avant-window-navigator-trunk avant-window-navigator-data-trunk python-awn-trunk awn-settings-trunk awn-applets-python-core-trunk python-awn-extras-trunk awn-applets-python-extras-trunk awn-applets-c-core-trunk awn-applets-c-extras-trun

Ahora un par de herramientas para aquellos leechers como yo que se acostumbran a descargar muchos archivos en descarga directa y partidos.

Jdownloader

Jdownloader es para mi el mejor gestor de descargas que existe, programado en Java. Para instalarlo en Ubuntu simplemente abrimos la terminal:

sudo add-apt-repository ppa:jd-team/jdownloader
sudo apt-get update
sudo apt-get install jdownloader

Hacha en Ubuntu? ¡No! -> Dalle

Pese a que Jdownloader integra un sistema para unir archivos no soporta cualquier extensión de éstos, asi que seguramente necesitaremos un programa alternativo. Yo propongo Dalle. Es un programa genial para partir y unir archivos. Tiene una GUI parecida a Hacha y acepta extensiones de multitud de sistemas, entre ellos: Astrotite, Axman 3, Easy File Splitter, File Splitter, Hacha (1, 2 and Pro), SplitFile, Zip…

Viene en un paquete .deb para que sea sencillisimo de instalar:

http://sourceforge.net/projects/dalle/files/dalle/dalle-0.10.10/dalle_0.10.10-1_all.deb/download
